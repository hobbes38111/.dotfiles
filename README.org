* To backup
** OSX
Run `./osx/save-hotkeys.sh` to backup your custom OSX hotkeys.
This will create a `osx/apply-keys.sh`s script.

** IntelliJ IDEA
Export editor settings and move them here.

Run `./install`

Manually import the settings
